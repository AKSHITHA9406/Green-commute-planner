**Course:** Software Engineering  
**University:** Mahindra University

---

## 📖 Introduction
The Green Commute Planner is a web-based software solution designed to encourage environmentally sustainable commuting. It helps users choose eco-friendly transportation options such as public transport, carpooling, cycling, and walking.

The system aims to reduce carbon emissions, traffic congestion, and fuel consumption while promoting greener travel habits.

---

## 🎯 Objectives
- Promote eco-friendly transportation choices  
- Calculate and compare carbon emissions  
- Provide optimal routes based on sustainability and convenience  
- Raise awareness about environmental impact  
- Develop a user-friendly and scalable application  

---

## 🧭 Scope of Work
The application is developed as a web platform where users can plan daily commutes by analyzing:
- Distance  
- Time  
- Cost  
- Estimated CO₂ emissions  

The system recommends the most sustainable travel options.
